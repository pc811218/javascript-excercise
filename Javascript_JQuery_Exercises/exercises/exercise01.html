請實做一簡易卡片產生器：

  • 當選擇樣式1、2、3時，卡片會分別套用xStyle、yStyle、zStyle
  • 當按下"鎖定文字"按鈕時，會將"卡片1文字"框Disabled，同時將"卡片1文字"框內容寫入卡片一，
    按鈕文字變為"解除鎖定"
  • 當按下"解除鎖定"，才可重新編輯文字框
  • 按下變更框線，卡片1可以在紅黑兩種框線顏色中切換
  • 按下卡片複製，會將卡片1的所有屬性及文字複製一份至卡片2

    • Hint: Class Attribute

    卡片1樣式： 樣式1　 樣式2　 樣式3　
    卡片1文字：
    卡片1
    卡片2
